# AniVerseX Spring Boot Assignment

Simple Spring Boot MVC + CRUD module for anime recommendations and reviews.

## Run
1. Create database:
```sql
CREATE DATABASE aniversex_db;
```
2. Edit `src/main/resources/application.properties` for DB username/password.
3. Run:
```bash
mvn spring-boot:run
```
4. Open: `http://localhost:8080/`
