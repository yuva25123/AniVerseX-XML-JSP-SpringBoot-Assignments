# AniVerseX - Assignment 2 Collection

This folder contains 8 Web Programming assignments under one common theme:
**AniVerseX - Anime Recommendations and Reviews**.

## Folder Plan

1. `01-springboot-crud`: Spring Boot MVC + CRUD for anime and reviews (code included).
2. `02-xml-dtd`: XML catalog with DTD-based validation for anime data (starter files included).
3. `03-xpath`: XPath queries on anime XML documents (README placeholder).
4. `04-xquery`: XQuery examples for anime recommendation extraction (README placeholder).
5. `05-xml-schema`: XML Schema (XSD) validation for anime catalog XML (starter XML included).
6. `06-xml-parser`: XML parsing in Java (DOM/SAX basic parser) for anime data (README placeholder).
7. `07-xml-java-jsp`: JSP-based UI integrated with XML/Java basics (starter JSP files included).
8. `08-jsp-jdbc-springboot`: Combined JSP + JDBC + Spring Boot integration assignment (README placeholder).
